About

MySideEffects is a step towards making information from the publications more accessible and readable.

It is built collaboratively as a part of the ‘Hackathon’ event held at Basel from 28 October , 2017 to 29 October, 2017.

It contains information about the drugs and their adverse effects. The user can have access to the information he/she is interested in.

Note: The information is going to be more accurate as we have more information about the user.

Disclaimer: This is still only collected statistcs, and everything can be different in your specific case, primarily for situations we are unable to identify.

Data resources:

OpenFDA Adverse event data - OpenFDA

PubMed (MEDLINE) - API for bulk download: PubMed

SIDER (side effect database, extracted from public documents and package inserts, last update: October 2015) - SIDER

MEDRA (Dictionary of medical terms): MEDRA

PUBCHEM (Database of drug compounds): PUBCHEM 1 PUBCHEM 2

This is part of the [BaselHack](baselhack.ch) Hackathon: Github
